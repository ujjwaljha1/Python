
1. Q1-What is the correct way to access the last element of a Python list?
   A. list[-1]
   B. list(1)
   C. list[last]
   D. list.end()
   **Correct Answer:** A. list[-1]

2. Q2-Which of the following methods is used to add an element to the end of a Python list?
   A. list.append()
   B. list.insert()
   C. list.add()
   D. list.extend()
   **Correct Answer:** A. list.append()

3. Q3-How do you access an element in a specific row and column of a multidimensional list in Python?
   A. list[row, column]
   B. list(column)(row)
   C. list(column, row)
   D. list[row][column]
   **Correct Answer:** D. list[row][column]

4. Q4-What is the purpose of list comprehension in Python?
   A. To create a new list by modifying the existing list.
   B. To perform complex mathematical computations on lists.
   C. To check if a list is empty.
   D. To reverse the order of elements in a list.
   **Correct Answer:** A. To create a new list by modifying the existing list.

5. Q5-How do you use list comprehension to create a list of squares for even numbers from 1 to 10?
   A. [x for x in range(1, 11) if x % 2 == 0]
   B. [x ** 2 for x in range(1, 11)]
   C. [x for x in range(1, 11) if x % 2 != 0]
   D. [x ** 2 for x in range(1, 11) if x % 2 == 0]
   **Correct Answer:** D. [x ** 2 for x in range(1, 11) if x % 2 == 0]

6. Q6-How do you use dictionary comprehension in Python to create a dictionary with the squares of numbers from 1 to 5?
   A. {x: x for x in range(1, 6)}
   B. {x: x ** 2 for x in range(1, 5)}
   C. {x ** 2: x for x in range(1, 6)}
   D. {x: x ** 2 for x in range(1, 6)}
   **Correct Answer:** D. {x: x ** 2 for x in range(1, 6)}

8. Q8-How do you access the value associated with a specific key in a Python dictionary?
   A. Using the get() method
   B. Using the key index
   C. Using the value() function
   D. Using the retrieve() method
   **Correct Answer:** A. Using the get() method

9. Q9-What is the key characteristic of a Python set?
   A. Sets do not allow duplicate elements.
   B. Sets are ordered collections.
   C. Elements in a set are accessed using indices.
   D. Sets can only contain numeric values.
   **Correct Answer:** A. Sets do not allow duplicate elements.
